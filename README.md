# Heart AI System 🫀

Welcome to the **Heart AI System** repository! This project leverages Artificial Intelligence to analyze health data and provide insights related to heart conditions.

## 🚀 About the Project
This system uses a machine learning model (`heart_model.pkl`) to predict or analyze heart-related issues. It features a React-based frontend and a Python backend.

Our goal is to create an easy-to-use health analysis tool that can assist users and medical professionals.

## 🛠️ Tech Stack
- **Frontend**: React (Vite)
- **Backend**: Python (Flask / FastAPI)
- **Machine Learning**: Scikit-Learn
- **Deployment**: Render

## 🏃‍♂️ Running Locally
1. Clone the repository: `git clone <your-repo-url>`
2. Setup Backend:
   - Navigate to the backend directory
   - Install requirements: `pip install -r requirements.txt`
   - Run the backend server
3. Setup Frontend:
   - Navigate to the frontend directory
   - Install dependencies: `npm install`
   - Run the frontend: `npm run dev`
